# GoRestaurant_Mobile

<img src="./src/assets/print.png" />

Desafio Gostack 11 utilizando React-Native e Typescript consumindo API REST com métodos CRUD.

Intruções:

1- Baixar dependências: 
```javascript
 yarn
```

2- Executar Fake API:
```javascript
   yarn json-server server.json -p 3333
```


3- Executar App:
```javascript
   yarn ios
```
